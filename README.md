# 系统管理

## 菜单管理

---

> 1. 列表字段：菜单名称，路由地址（前端路由），菜单图标，菜单编号，排序优先级
> 2. 列表使用树状列表显示，默认只显示顶级菜单，点击菜单将展开其下级菜单

### 查询

筛选字段：菜单名称，菜单编号

### 新增

表单字段：*菜单名称，上级菜单，*路由地址，*菜单图标，*菜单编号，*菜单类型【菜单，按钮】，*排序优先级，菜单备注

### 修改

表单字段：同上

### 删除

允许级联删除 - 角色管理

## 角色管理

---

> 1. 列表字段：角色名称，职级，被绑定的用户个数，排序优先级
> 2. 所有角色都需要绑定 “角色职级” 【Admin，辅导员，普通教师，学生】（'ADMIN','DEAN','SECOND_DEAN','COUNSELOR','HEAD_TEACHER','TEACHER','STUDENT'）
> 3. 所有角色都需要指定 “上级角色”（Admin 除外）
> 4. 列表使用树状列表显示，默认只显示顶级角色（如 Admin），点击角色将展开其下级角色

### 查询

筛选字段：角色名称，角色职级

### 新增

表单字段：*角色名称，上级角色，*职级，*排序优先级

### 修改

表单字段：同上

### 删除

删除角色需同时满足以下条件：

1. 未被其他角色绑定为 “上级角色”
2. “被绑定的用户个数” 为 0

### 设置角色的权限

选择一个角色后，允许勾选 “菜单管理” 的所有项（每个菜单信息都对应一个权限）
